Introduction to CSS: A Medieval Adventure

What is CSS?

CSS stands for Cascading Style Sheets. It is a language used to style HTML documents and control the appearance of web pages. With CSS, you can add colors, fonts, backgrounds, borders, and other decorative elements to your web page.

How does CSS work?

CSS works by targeting HTML elements and applying styles to them. You can target elements by their tag name, class name, or ID. For example: 

				
					p {
					    color: red;
					}
					
					.header {
					    font-size: 24px;
					    text-align: center;
					}
					
					#main {
					    background-color: #fff;
					    border: 1px solid black;
					    padding: 10px;
					}

In this example, the first rule targets all p elements on the page and sets their color to red. The second rule targets elements with the class name "header" and sets their font size to 24 pixels and text alignment to center. The third rule targets an element with the ID "main" and sets its background color to white, border to 1 pixel solid black, and padding to 10 pixels.
